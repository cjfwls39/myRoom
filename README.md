단순한 이력서 형식에서 벗어나, 3D 공간을 직접 탐색하며 포트폴리오를 경험할 수 있는 인터랙티브 사이트입니다.  
방 안의 오브젝트와 상호작용하며 저를 소개합니다.

*A portfolio site where you don't just read — you explore.  
Navigate a handcrafted 3D room and interact with objects to discover who I am.*

## ✨ Features

| 기능 | 설명 |
|------|------|
| 🏠 **3D 인터랙티브 룸** | 마우스 드래그·휠로 방 전체를 자유롭게 탐색 |
| 🎬 **카메라 인트로** | 접속 시 카메라가 방 안으로 부드럽게 진입하는 연출 |
| 🌙 **낮/밤 전환** | 조명·색상이 실시간으로 부드럽게 전환 |
| ❄️ **날씨 모드** | 스노우글로브 배경으로 전환되는 블리자드 씬 |
| 🖱️ **오브젝트 호버** | 오브젝트에 마우스를 올리면 반응하는 애니메이션 |
| 🪟 **포트폴리오 모달** | 오브젝트 클릭 시 About / Projects / Skills / Contact 팝업 |
| 🕐 **실시간 시계** | 방 안 벽시계가 실제 현재 시각을 표시 |

## 🚀 Getting Started

```bash
# 1. 저장소 클론 / Clone
git clone https://github.com/cjfwls39/myRoom.git
cd myRoom

# 2. 의존성 설치 / Install
npm install

# 3. 로컬 서버 실행 / Run dev server
npm run dev
```

> ⚠️ WebGL을 지원하는 최신 브라우저(Chrome / Firefox / Edge)에서 접속을 권장합니다.  
> ⚠️ A WebGL-compatible modern browser is required.
